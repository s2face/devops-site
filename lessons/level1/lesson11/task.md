### H2: apt: работа с пакетами в Ubuntu/Debian
```bash
sudo apt update              # обновить индекс
sudo apt upgrade -y          # обновить пакеты
sudo apt install nginx -y    # установить пакет
sudo apt remove nginx        # удалить
sudo apt search "web server" # поиск
```
