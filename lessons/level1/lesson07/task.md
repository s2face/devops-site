Читатель научится создавать ветки, переключаться между ними, делать слияния, решать конфликты и познакомится с популярными стратегиями ветвления.

### H2: Создание и переключение между ветками
```bash
git branch feature/login
git switch feature/login
# или одной командой:
git switch -c feature/login
```
### H2: Слияние веток: git merge
```bash
git switch main
git merge feature/login
```
