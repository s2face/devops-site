## Цель урока
Читатель освоит базовые команды Bash для навигации по файловой системе, работы с файлами и каталогами, просмотра процессов и работы с текстом.

## Целевая аудитория
Новички, только установившие Ubuntu Server (урок 2).

## Пререквизиты
- Урок 2: рабочий Ubuntu Server, доступ по SSH

## Технический стек
- **Shell:** Bash 5.x
- **OS:** Ubuntu Server 22.04

## Требования к контенту
- [ ] Анатомия командной строки: prompt, команда, флаги, аргументы
- [ ] Навигация: `pwd`, `ls`, `cd`, `tree`
- [ ] Работа с файлами: `touch`, `cp`, `mv`, `rm`, `mkdir`, `rmdir`
- [ ] Просмотр содержимого: `cat`, `less`, `head`, `tail`
- [ ] Поиск: `find`, `grep`
- [ ] Работа с текстом: `echo`, `wc`, перенаправление `>`, `>>`, `|`
- [ ] Процессы: `ps`, `top`, `kill`
- [ ] Помощь: `man`, `--help`
