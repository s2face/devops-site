## Цель урока
Читатель поймёт структуру файловой системы Linux (FHS), научится работать с дисками, монтированием и архивами.

## Целевая аудитория
Начинающие системные администраторы / DevOps-инженеры.

## Пререквизиты
- Уроки 3–4: Bash и работа с правами

## Технический стек
- **Команды:** `df`, `du`, `mount`, `umount`, `fdisk`, `lsblk`, `tar`, `gzip`, `zip`, `rsync`
- **Файлы конфигурации:** `/etc/fstab`

## Требования к контенту
- [ ] Стандарт FHS: /bin, /etc, /var, /tmp, /home, /usr, /proc, /sys
- [ ] Просмотр дискового пространства: `df -h`, `du -sh`
- [ ] Просмотр блочных устройств: `lsblk`, `fdisk -l`
- [ ] Монтирование: `mount`, `umount`, `/etc/fstab`
- [ ] Создание архивов: `tar`, `gzip`, `bzip2`, `zip`
- [ ] Распаковка архивов
- [ ] `rsync` для синхронизации файлов
- [ ] Символические и жёсткие ссылки: `ln`
