## Цель урока
Читатель установит Ubuntu Server 22.04 LTS в VirtualBox или на реальное железо, выполнит первичную настройку системы и будет готов к дальнейшей работе в терминале.

## Целевая аудитория
Новички, знакомые с Windows. Первый опыт с Linux.

## Пререквизиты
- Урок 1 (понимание зачем нужен Linux в DevOps)
- Установленный VirtualBox или доступ к VPS

## Технический стек
- **ОС:** Ubuntu Server 22.04 LTS
- **Виртуализация:** VirtualBox 7.x (альтернатива: VMware, облачный VPS)
- **Минимальные требования:** 2 CPU, 2 GB RAM, 20 GB HDD

## Требования к контенту
- [ ] Объяснить разницу между Ubuntu Desktop и Ubuntu Server
- [ ] Пошаговая установка в VirtualBox (скриншоты описать текстово)
- [ ] Первичная настройка: hostname, timezone, локаль
- [ ] Создание нового пользователя и выдача sudo
- [ ] Обновление пакетов после установки
- [ ] Настройка сети (статический IP или DHCP)
- [ ] Установка openssh-server для удалённого подключения

```bash
# Обновление системы
sudo apt update && sudo apt upgrade -y

# Настройка hostname
sudo hostnamectl set-hostname devops-server

# Установка SSH-сервера
sudo apt install -y openssh-server
sudo systemctl enable ssh
```
